📌 Simple Validation Form

📝 Description
This is a simple and responsive form validation project built using HTML, CSS, and JavaScript.
It validates user inputs in real-time and ensures that all required fields are correctly filled before submission.
The project demonstrates basic frontend form handling and client-side validation techniques.

🚀 Features
Real-time input validation
Required field checks
Email format validation
Password length validation
User-friendly error messages
Clean and responsive UI

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

⚙️ How It Works
User fills out the form fields
JavaScript checks each input for valid data
If any input is invalid, an error message is shown
Form is only submitted when all validations pass
